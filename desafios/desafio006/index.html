Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir.Passe o mouse sobre os termos para ver qual tag deverá utilizar em cada caso.

  • Parágrafo

    • Usamos a 'tag p'

  • Títulos

    • Usamos a 'tag h1'

  • Imagens

    • Usamos a 'tag img'

  • Abreviações

    • Usamos a Tag abbr

  • Códigos fonte

    • Usamos a tag code

  • Listas enumeradas

    • Usamos a 'tag ol'

  • Lista com marcadores

    • Usamos a 'Tag ul'
